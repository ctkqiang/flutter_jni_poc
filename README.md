# Flutter JNI 漏洞测试工具

这是一个基于 Flutter 和原生 Android 的 JNI 漏洞测试工具，用于测试和验证 JNI 指针操作的安全性。

## 功能特性

- **原生 Android UI**：使用 Android 原生布局和控件，提供流畅的用户体验
- **多种指针测试**：
  - 有效指针测试：使用真实的 AndroidShellHolder 指针
  - 伪造指针测试：使用固定的伪造指针 0x12345678L
  - 随机指针测试：每次生成随机的指针值
  - 序列测试：自动测试多种指针值，包括空指针、最小指针、最大指针等
  - 内存测试：在 while 循环中不断执行伪造指针测试，直到达到最大内存
- **长按自动执行**：伪造指针按钮支持长按自动执行功能
- **详细的日志输出**：使用 logcat 记录所有操作和结果
- **内存使用监控**：实时显示内存使用情况
- **操作统计**：记录操作数、成功数和失败数
- **SafeArea 支持**：确保 UI 内容不会被系统 UI 遮挡

## 安装方法

1. 克隆项目到本地：
   ```bash
   git clone <repository_url>
   cd flutter_jni_poc
   ```

2. 安装依赖：
   ```bash
   flutter pub get
   ```

3. 构建并运行：
   ```bash
   flutter run --release
   ```

## 使用说明

1. **有效指针**按钮：测试使用真实的 AndroidShellHolder 指针
2. **伪造指针**按钮：测试使用固定的伪造指针 0x12345678L，长按可自动重复执行
3. **随机指针**按钮：测试使用随机生成的指针值
4. **序列测试**按钮：自动测试多种指针值，包括空指针、最小指针、最大指针等
5. **内存测试**按钮：在 while 循环中不断执行伪造指针测试，直到达到最大内存
6. **重置统计**菜单项：重置操作统计信息

## 日志查看

使用以下命令查看详细日志：
```bash
adb logcat | grep JNI_VULNERABILITY_TEST
```

## 技术实现

- **Flutter 框架**：作为应用的基础框架
- **原生 Android UI**：使用 LinearLayout、TextView、Button 等原生控件
- **JNI 接口**：通过 JNI 调用原生 C++ 代码进行指针操作
- **多线程处理**：使用 Thread 进行后台操作，避免阻塞 UI
- **日志系统**：使用 Android Log 类记录详细日志

## 注意事项

- 本工具仅用于测试和验证 JNI 指针操作的安全性，请勿用于恶意用途
- 内存测试可能会导致设备内存使用量增加，请谨慎使用
- 在测试过程中，可能会出现应用崩溃或设备重启的情况，这是正常现象
